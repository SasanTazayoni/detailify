## Setup & Running Locally

Follow these steps to run the project locally:

### Prerequisites

- **Node.js v18 or later**
- **npm** (comes with Node)

1. Type ```git clone https://github.com/SasanTazayoni/detailify.git``` in the terminal.
2. Change directories into the folder with ```cd detailify```.
3. Download [Node](https://nodejs.org/).
4. Install the dependencies with ```npm install```.
5. Start the development server with ```npm run dev```.
